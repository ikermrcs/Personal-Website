ABOUT ME

Navigating these complexities and contributing to the transformation of the industry is what truly excites me.

PYTHON PROJECTS

Monte Carlo Simulation

Conducted a project analyzing the impact of climate on cocoa production in Ivory Coast and, afterwards, the impact of global cocoa production on the price of cocoa futures. The objective was to become familiar with data analysis and linear regressions in Python.

Developed a tool that allows the user to select a random number of stocks and create a portfolio with random weights. The program calculates the return of the stocks and the portfolio over given time periods and compares them to the S&P500.

Designed a Python program to simulate casino roulette spins, tracking the percentage of hits for each color (red, black, green) and the performance of a virtual wallet. The simulator demonstrates how probabilities stabilize over the long term and highlights the impossibility of winning in a casino, even with favorable odds.
